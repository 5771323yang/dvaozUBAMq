# 学生宿舍管理系统 java679

## 项目概述

学生宿舍管理系统是基于Spring Boot和Vue的前后端分离项目。系统采用B/S架构，通过结合Spring Boot、Mybatis、Vue、Element-UI、MySQL、Redis、Maven、Pinia等技术开发，致力于提高宿舍管理的效率和便捷性。

## 技术栈

- **后端**：Spring Boot、Mybatis、Redis
- **前端**：Vue、Element-UI、Pinia
- **数据库**：MySQL
- **构建工具**：Maven

## 功能模块

- 学生信息管理
- 宿舍楼信息管理
- 宿舍房间信息管理
- 维修申报处理
- 缴费管理
- 安全管理

## 系统角色

- 系统管理员
- 学生用户
- 宿舍管理员

## 运行环境

- **后端**：JDK 1.8 或以上版本
- **数据库**：MySQL 5.7 或以上版本
- **前端**：现代浏览器
- **操作系统**：Windows/Linux/Mac OS

## 部署步骤

1. 准备好运行环境，包括Java环境和数据库环境。
2. 导入MySQL数据库。
3. 使用Maven构建后端工程。
4. 部署前端工程。
5. 启动后端服务。
6. 打开前端页面，进行访问。

## 核心亮点

- **前后端分离**：提高开发效率和维护性。
- **界面简洁**：提供直观、易用的用户操作界面。
- **高效率**：基于Spring Boot和Vue，实现快速的数据处理和响应。
- **数据存储**：使用MySQL数据库确保数据的安全性和一致性。
- **缓存技术**：Redis的应用加快了系统的响应速度。
